# GENESIS FURY ·《创世之怒》— 设计文档

> Populous 系上帝游戏（God Game）。玩家扮演苍蓝之神，不直接操控单位，
> 通过塑造地形与施放神迹，引导信徒繁衍扩张，消灭绯红邪神的信众。

## 1. 核心循环

```
信徒定居 → 建屋升级 → 人口增长 → 产出信仰(法力)
   ↑                                    ↓
领土扩张 ← 地形平整/庇护 ← 神迹(塑地/祝福/天罚) ←┘
```

- **信徒 (Follower)**：自主 AI。寻找己方领土附近的平地 → 定居建屋；无地可建则游荡/朝圣。
  遭遇敌方信徒进入战斗。玩家不能直接下令，只能用「集结图腾」引导走向。
- **信仰 (Faith)**：法力资源。`regen = 2 + 0.55 × 人口`（/秒），上限 `200 + 12 × 人口`。
- **神迹 (Miracle)**：唯一的直接干预手段，全部消耗信仰（见 §4）。
- **领土**：以己方建筑为核心的影响圈。领土越大 → 可定居点越多 → 人口上限越高。

## 2. 胜负条件

| 条件 | 结果 |
|------|------|
| 敌方人口（信徒+屋内居民）归零 | **胜利** — 结算画面（用时/峰值人口/神迹次数/评级 S~C） |
| 己方人口归零 | **失败** |
| 20 分钟后进入「终焉审判」 | 双方信仰产出×3，火山/洪水费用减半，强制决战 |

## 3. 地形系统

- **数据**：129×129 角点高度场（int 0..14），对应 128×128 瓦片。水位 `WL=2`。
- **渲染**：16×16 瓦片一个 chunk 的 WebGL Mesh，角点高度位移 + 逐顶点光照（坡度朝阳面亮、背阴面暗）。地形变化只重建受影响 chunk。
- **瓦片分类**（由平均高度+坡度）：深水 / 浅水 / 沙滩 / 草原 / 荒岩(坡度陡) / 雪峰(h≥11)。
- **可建造判定**：4 角等高、非水、非沼泽、坡度 0 → 可建屋。周边 5×5 内平地数决定房屋等级上限。
- **生成**：种子化值噪声（fBm）+ 中央大陆掩膜 + 双方出生高地，保证开局对称公平。

| 神迹 | 费用 | 冷却 | 效果 | 反馈链 |
|------|-----:|-----:|------|--------|
| 塑地·隆起 Raise | 8 | — | 笔刷中心 +1 高度（半径2高斯衰减） | 预瞄环→石鸣→尘土粒子→地面隆起动画 |
| 塑地·沉降 Lower | 8 | — | 同上 −1，可淹没敌地 | 同上，下沉音 |

| 雷罚 Lightning | 60 | 3s | 单点：即死 1-2 单位，木屋起火 | 蓄能辉光→霹雳闪→分叉电弧→焦痕 |
| 沼泽 Swamp | 70 | 10s | r=3 沼泽地 25s：困住并缓杀路过单位 | 绿雾→咕噜声→气泡粒子 |
| 地震 Earthquake | 120 | 20s | r=6：高度随机抖动 ±2，建筑 60% 坍塌 | 屏幕震动→轰鸣→裂缝→尘暴 |
| 洪水 Flood | 200 | 45s | 全图水位 +1，25s 后退去；低地文明灭顶 | 号角→海面上涨动画→溺水音 |

## 5. 信徒 / 建筑数值

| 项 | 值 |
|----|----|
| 信徒 HP / 攻击 | 10 / 1.2 dps（战斗为接触判定） |
| 移动速度 | 1.6 tile/s（沼泽 ×0.25） |
| 房屋等级 1/2/3 | 容量 2/4/7，产出信仰加成 ×1/×1.15/×1.3，HP 30/55/90 |
| 繁衍 | 每屋每 `9/7/5.5s`（按等级）生成 1 信徒，直至容量满；全局人口上限 = Σ房屋容量 + 8 |
| 出屋参战 | 房屋被打/邻近战斗时居民出屋防御 |
| 火焰 | 起火房屋 6s 内塌；可被祝福扑灭 |

## 6. 敌方 AI（绯红邪神）

状态机，每 1.2s 决策一次（难度调频率：简单 2.2s / 普通 1.2s / 困难 0.7s）：

1. **EXPAND**（默认）：己方领土边缘找近水平缓区 → 塑地整平造可建地；信徒自然定居。
2. **HARASS**（信仰 > 40%上限）：对玩家人口密度最高区按权重选 雷罚(50%)/沼泽(30%)/地震(20%)。
3. **DEFEND**（近 8s 己方损失 > 3 单位）：对受创区祝福；若敌军压境放图腾回撤。
4. **CATACLYSM**（信仰 > 85%上限 且 人口领先）：火山砸玩家主城；若落后改洪水（自家在高地时）。
5. 失误率：简单 35% 决策空转；困难 5%。

## 7. 操作 / 手感

- 鼠标：左键施法、右键/中键拖拽平移、滚轮缩放（指向缩放）、屏幕边缘滚动；WASD/方向键平移。
- 触屏：单指拖动平移、双指捏合缩放、点击施法；HUD 按钮加大热区。
- 镜头：目标插值缓动（临界阻尼），拖拽带惯性；地震/火山按距离衰减屏幕震动。
- 全神迹反馈链：预瞄指示（范围环+可行性着色）→ 施法音 → 粒子/光效 → 地形/单位反应。

## 8. 技术架构

```
src/
  core/    常量表·事件总线·种子RNG·存档(localStorage)
  sim/     确定性模拟: world(高度场) followers buildings miracles ai sim(固定步长10Hz)
  render/  PixiJS v8: terrainMesh(chunk) water sprites fx camera minimap
  input/   pointer/keyboard/touch 统一手势
  audio/   WebAudio 全程序化: 环境音景 + 生成式配乐 + 合成SFX
  ui/      DOM 主题化 HUD + 屏幕流(标题/载入/设置/教学/暂停/结算)
  assets/  manifest 加载器 + 程序化兜底
```

- 模拟固定步长 10Hz 与渲染 60fps 解耦（渲染插值单位位置）。
- 性能：chunk mesh 缓存、单位精灵池、粒子池、可视区裁剪、按需重建。
- 存档：sim 全量序列化（高度场 RLE）+ 设置持久化；页面刷新可续局。

## 9. 美术方向

关键词锁定：*painterly epic fantasy, ancient mythic civilization, warm golden-hour light,
weathered stone & moss & aged parchment, soft volumetric clouds, muted teal vs crimson factions*。
全部素材由 gpt-image 系模型生成（见 docs/ASSETS.md），sharp 后处理，程序化兜底保证无破图。
